🖥️ Console Challenges

Deschide DevTools (F12 → Console) și scrie cod JavaScript real pentru a manipula această pagină. Învață făcând!

Apasă F12 sau Ctrl+Shift+J → tab-ul Console

🏆 TOATE 9 COMPLETATE! Ești un MAESTRU al consolei! 🏆
1
Bifează checkbox-ul
querySelector + .checked
Folosește querySelector pentru a găsi checkbox-ul și setează-i proprietatea checked pe true.
// Scrie asta în consolă: document.querySelector('#task1-check').checked = true;
⏳ În așteptare
2
Schimbă textul
getElementById + textContent
Găsește elementul cu getElementById și schimbă-i textul folosind textContent.
// Schimbă textul: document.getElementById('task2-text').textContent = 'Am reușit!';
Schimbă-mă din consolă!
⏳ În așteptare
3
Adaugă o clasă CSS
classList.add()
Adaugă clasa active pe cutie. CSS-ul va face restul — culoare, rotire, umbră!
// Adaugă clasa 'active': document.querySelector('#task3-box').classList.add('active');
⏳ În așteptare
4
Schimbă stilul CSS
element.style.property
Modifică background-ul cutiei la orice culoare vrei (ex: cyan, #ff6b6b, gold).
// Schimbă background-ul: document.getElementById('task4-box').style.background = 'cyan';
⏳ În așteptare
5
Adaugă un Event Listener
addEventListener('click', ...)
Adaugă un click listener pe buton care schimbă textul. Apoi apasă butonul!
// Adaugă listener, apoi click pe buton: document.getElementById('task5-btn').addEventListener('click', function() { this.textContent = '✅ Funcționează!'; });
⏳ În așteptare
6
Creează un element nou
createElement + appendChild
Creează un element <div> cu text și adaugă-l în container.
// Creează și adaugă un element: var el = document.createElement('div'); el.textContent = 'Nou element creat! 🎯'; el.style.cssText = 'padding:8px;background:#00e676;color:#000;border-radius:8px;font-weight:600'; document.getElementById('task6-container').appendChild(el);
Container gol — adaugă ceva!
⏳ În așteptare
7
Modifică un atribut
setAttribute()
Schimbă placeholder-ul input-ului folosind setAttribute.
// Schimbă placeholder-ul: document.getElementById('task7-input').setAttribute('placeholder', 'Schimbat din consolă! 🚀');
⏳ În așteptare
8
Injectează HTML
innerHTML
Folosește innerHTML pentru a adăuga conținut HTML cu tag-uri în container.
// Injectează HTML: document.getElementById('task8-container').innerHTML = '<strong style="color:#00d2ff">🔥 HTML injectat!</strong>';
Conținut HTML va apărea aici...
⏳ În așteptare
9
Șterge un element
element.remove()
Folosește remove() pentru a șterge elementul roșu din pagină. Dispare complet!
// Șterge elementul: document.getElementById('task9-target').remove();
🗑️ Șterge-mă din consolă!
