Provocare 1

☑️ Checkbox Magic — Selectează 3 și vezi ce se întâmplă

🎯 TASK: Scrie în consolă cod care:
1. Ascultă TOATE checkboxurile cu addEventListener("change")
2. Numără câte sunt bifate (:checked)
3. Dacă sunt 3 sau mai multe bifate → schimbă textul din #checkboxStatus în "🎉 Ai deblocat secretul!" cu culoare verde
4. Dacă sunt mai puțin de 3 → pune textul înapoi la normal
Selectează 3 mâncăruri preferate... 🔒
// Scrie asta în consolă (F12): document.querySelectorAll(".magic-cb").forEach(cb => { cb.addEventListener("change", () => { const checked = document.querySelectorAll(".magic-cb:checked").length; const status = document.getElementById("checkboxStatus"); if (checked >= 3) { status.textContent = `🎉 Ai deblocat secretul! (${checked} selectate)`; status.style.color = "#00e676"; status.style.fontSize = "1.2rem"; } else { status.textContent = `Selectează ${3 - checked} mai multe... 🔒`; status.style.color = ""; status.style.fontSize = ""; } }); });

📊 Character Counter — Numără în timp real

🎯 TASK: Scrie în consolă cod care:
1. Ascultă "input" pe #charInput
2. La fiecare keystroke, afișează numărul de caractere în #charCount
3. Umple bara de progres #charBarFill proporțional (max = 50 caractere)
4. La 50+ caractere, fă #charCount roșu și arată "PREA MULT!"
0 / 50 caractere
// Scrie asta în consolă: document.getElementById("charInput").addEventListener("input", (e) => { const len = e.target.value.length; const max = 50; const counter = document.getElementById("charCount"); const fill = document.getElementById("charBarFill"); counter.textContent = `${len} / ${max} caractere`; fill.style.width = `${Math.min((len / max) * 100, 100)}%`; if (len > max) { counter.textContent = `⚠️ PREA MULT! (${len})`; counter.style.color = "#e94560"; fill.style.background = "#e94560"; } else { counter.style.color = ""; fill.style.background = ""; } });
Provocare 4

🛒 Shopping List — Adaugă și Șterge

🎯 TASK: Scrie în consolă cod care:
1. La click pe #shopAdd → ia textul din #shopInput și adaugă un <li> în #shopList
2. Enter în input = la fel ca click pe buton
3. La click pe un item din listă → șterge-l (removeChild)
4. Golește inputul după adăugare și dă-i focus
  • 🥛 Lapte — click ca să ștergi
  • 🍞 Pâine — click ca să ștergi
// Scrie asta în consolă: function addItem() { const input = document.getElementById("shopInput"); const text = input.value.trim(); if (!text) return; const li = document.createElement("li"); li.textContent = text; // Click pe item = șterge li.addEventListener("click", () => li.remove()); document.getElementById("shopList").appendChild(li); input.value = ""; input.focus(); } document.getElementById("shopAdd").addEventListener("click", addItem); document.getElementById("shopInput").addEventListener("keydown", (e) => { if (e.key === "Enter") addItem(); }); // Fă și item-urile existente clicabile document.querySelectorAll("#shopList li").forEach(li => { li.addEventListener("click", () => li.remove()); });

🏆 Score Tracker — Plus și Minus

🎯 TASK: Scrie în consolă cod care:
1. Click pe #plusBtn → mărește scorul cu 1
2. Click pe #minusBtn → micșorează scorul cu 1
3. Click pe #resetBtn → resetează la 0
4. Scorul negativ = roșu, pozitiv = verde, zero = normal
5. BONUS: La scorul 10+ → afișează mesaj "🏆 CAMPION!" în #scoreMsg
0
Folosește butoanele... dacă funcționează 😏
// Scrie asta în consolă: let score = 0; function updateScore() { const display = document.getElementById("scoreValue"); const msg = document.getElementById("scoreMsg"); display.textContent = score; if (score > 0) { display.style.color = "#00e676"; } else if (score < 0) { display.style.color="#e94560" ; } else { display.style.color="#00d2ff" ; } if (score>= 10) { msg.textContent = "🏆 CAMPION! Ai ajuns la " + score + "!"; msg.style.color = "#ffd43b"; } else if (score <= -5) { msg.textContent="💀 Ai intrat pe minus rău..." ; msg.style.color="#e94560" ; } else { msg.textContent=`Scor curent: ${score}`; msg.style.color="" ; } } document.getElementById("plusBtn").addEventListener("click", ()=> { score++; updateScore(); }); document.getElementById("minusBtn").addEventListener("click", () => { score--; updateScore(); }); document.getElementById("resetBtn").addEventListener("click", () => { score = 0; updateScore(); });

⏱️ Countdown Timer — setInterval + clearInterval

🎯 TASK: Scrie în consolă cod care:
1. Click pe #startTimer → pornește un countdown de la 10 (cu setInterval)
2. Afișează secundele în #timerDisplay, actualizat la fiecare secundă
3. La 0 → oprește timer-ul (clearInterval) și arată "⏰ TIMP EXPIRAT!"
4. Click pe #stopTimer → oprește timer-ul
5. Click pe #resetTimer → resetează la 10
6. BONUS: Sub 3 secunde → fă textul roșu și mai mare
10
// Scrie asta în consolă: let seconds = 10; let timerId = null; function updateTimerDisplay() { const display = document.getElementById("timerDisplay"); display.textContent = seconds; if (seconds <= 3 && seconds> 0) { display.style.color = "#e94560"; display.style.fontSize = "5rem"; } else if (seconds <= 0) { display.textContent="⏰ TIMP EXPIRAT!" ; display.style.fontSize="2rem" ; display.style.color="#e94560" ; clearInterval(timerId); timerId=null; } else { display.style.color="" ; display.style.fontSize="" ; } } document.getElementById("startTimer").addEventListener("click", ()=> { if (timerId) return; // deja rulează timerId = setInterval(() => { seconds--; updateTimerDisplay(); }, 1000); }); document.getElementById("stopTimer").addEventListener("click", () => { clearInterval(timerId); timerId = null; }); document.getElementById("resetTimer").addEventListener("click", () => { clearInterval(timerId); timerId = null; seconds = 10; updateTimerDisplay(); });
