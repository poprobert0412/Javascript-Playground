🔧 Chrome DevTools — Consolă Live

Învață să scrii JavaScript direct în consola browser-ului! Modifică orice pagină în timp real.

Interactiv • Practică pe această pagină!
🎓 Sfat de Prof
Această pagină e diferită — nu doar citești, ci faci! Deschide DevTools (F12) și scrie comenzile direct în consola Chrome pe această pagină. Vei modifica elementele LIVE! 🔥

⌨️ Cum deschizi DevTools

Pas 0

Deschide DevTools cu una din aceste combinații:

F12 Deschide DevTools
Ctrl + Shift + I DevTools
Ctrl + Shift + J Direct la Consolă

După ce se deschide, click pe tab-ul Console din partea de sus. Acolo vei scrie codul!

🟢 Nivel 1 — Primele comenzi în Consolă

Pas 1

📝 Scrie JavaScript direct în consolă

Deschide consola (F12 → Console) și scrie pe rând:

// Matematică simplă
2 + 2
10 * 5
Math.sqrt(144)

// Text
"salut".toUpperCase()
"JavaScript".length
"Robert".split("").reverse().join("")

// Variabile (le poți folosi cât e consola deschisă!)
let x = 42
x * 2
const nume = "Robert"
`Salut, ${nume}! Ai ${x} ani.`
💡 Tip
Consola Chrome e ca un REPL (Read-Eval-Print-Loop). Scrii cod → apesi Enter → vezi rezultatul instant!
Pas 2

🔍 console.log, console.warn, console.error

Nu doar console.log — ai și alte metode!

console.log("Mesaj normal")
console.warn("⚠️ Atenție la asta!")
console.error("❌ Ceva a mers prost!")
console.table([{name: "Ana", age: 22}, {name: "Ion", age: 25}])
console.time("test")
// ... cod ...
console.timeEnd("test") // → cât a durat

🟡 Nivel 2 — Modifică pagina LIVE

Pas 3

🎯 Selectează elemente de pe această pagină

Scrie în consolă pentru a selecta elementele de mai jos:

// Selectează box-ul de practică
const box = document.getElementById("practiceBox")
box // → vezi elementul

// Selectează titlul paginii
document.querySelector("h1").textContent

// Câte link-uri sunt în navbar?
document.querySelectorAll(".nav-links a").length

🎯 Zona de Practică — Modifică-mă din Consolă!

Eu sunt #practiceBox — schimbă-mi textul din consolă!
  • Item 1 — Schimbă-mi culoarea
  • Item 2 — Adaugă-mi un frate nou
  • Item 3 — Șterge-mă dacă poți!
Pas 4

✏️ Modifică textul și stilul

Copy-paste (sau scrie) aceste comenzi în consolă:

// Schimbă textul
document.getElementById("practiceBox").textContent = "Am fost modificat din consolă! 🎉"

// Schimbă culoarea
document.getElementById("practiceBox").style.color = "#00d2ff"
document.getElementById("practiceBox").style.fontSize = "1.5rem"
document.getElementById("practiceBox").style.border = "2px solid #e94560"

// Schimbă background-ul
document.getElementById("practiceBox").style.background = "linear-gradient(135deg, #1a1a2e, #16213e)"
Pas 5

➕ Adaugă și șterge elemente

// Creează un element NOU
const newItem = document.createElement("li")
newItem.textContent = "Item 4 — Creat din consolă! 🆕"
newItem.style.borderColor = "#00e676"
document.getElementById("practiceList").appendChild(newItem)

// Șterge ultimul element din listă
const list = document.getElementById("practiceList")
list.removeChild(list.lastElementChild)

// Adaugă HTML direct
document.getElementById("practiceBox").innerHTML = "<strong style='color:#e94560'>HTML injectat!</strong> 🔥"

🔴 Nivel 3 — Event Listeners Live

Pas 6

🎯 Adaugă un Event Listener din Consolă

Scrie în consolă pentru a face butonul interactiv:

// Adaugă click event pe buton
document.getElementById("practiceBtn").addEventListener("click", () => {
    alert("Ai dat click! 🎉 Asta e un event listener adăugat din consolă!")
})

// Numără click-urile
let clicks = 0
document.getElementById("practiceBtn").addEventListener("click", () => {
    clicks++
    document.getElementById("practiceBtn").textContent = `Clicks: ${clicks}`
})
Pas 7

⌨️ Ascultă input-ul în timp real

// Ascultă ce scrii în input
document.getElementById("practiceInput").addEventListener("input", (e) => {
    console.log("Ai scris:", e.target.value)
    document.getElementById("practiceBox").textContent = e.target.value
})

// Schimbă culoarea box-ului la fiecare keystroke
document.getElementById("practiceInput").addEventListener("input", (e) => {
    const r = Math.floor(Math.random() * 255)
    const g = Math.floor(Math.random() * 255)
    const b = Math.floor(Math.random() * 255)
    document.getElementById("practiceBox").style.borderColor = `rgb(${r},${g},${b})`
})
Pas 8

🖱️ Mouse events — hover, move, scroll

// Urmărește mouse-ul
document.addEventListener("mousemove", (e) => {
    document.title = `X: ${e.clientX} Y: ${e.clientY}`
})

// Schimbă toate item-urile la hover
document.querySelectorAll("#practiceList li").forEach(li => {
    li.addEventListener("mouseover", () => {
        li.style.background = "rgba(0, 210, 255, 0.15)"
        li.style.transform = "translateX(10px)"
    })
    li.addEventListener("mouseout", () => {
        li.style.background = ""
        li.style.transform = ""
    })
})

⚡ Nivel 4 — Trucuri Avansate DevTools

Pas 9

🔍 $ și $$ — Scurtături DevTools

// $ = querySelector (doar în consolă!)
$("h1")              // primul h1
$$("li")             // TOATE li-urile (array)
$("#practiceBox")   // element cu id

// $0 = ultimul element selectat în Elements tab!
// Click dreapta pe element → Inspect → apoi scrie $0 în consolă
$0.style.background = "red"

// Copiază tot textul unui element
copy($("h1").textContent) // → acum e în clipboard!
Pas 10

🕵️ Inspectează orice site!

Toate trucurile de aici funcționează pe orice pagină web! Încearcă pe site-urile tale preferate: 

// Pe orice site — schimbă titlul paginii
document.title = "Site-ul meu acum! 😎"

// Schimbă toate imaginile de pe o pagină
document.querySelectorAll("img").forEach(img => {
    img.style.borderRadius = "50%"
    img.style.transform = "rotate(180deg)"
})

// Fă totul editabil! (poți scrie direct pe pagină)
document.designMode = "on"
// Dezactivează cu: document.designMode = "off"

// Scoate toate culorile
document.querySelectorAll("*").forEach(el => {
    el.style.color = "black"
    el.style.background = "white"
})

// Rainbow mode!
let hue = 0
setInterval(() => {
    document.body.style.filter = `hue-rotate(${hue}deg)`
    hue += 5
}, 50)
⚠️ Important
Modificările din consolă sunt TEMPORARE — dispar la refresh (F5). Nu strici nimic permanent! E perfect pentru a experimenta.
Pas 11

📊 Network tab — Vezi ce încarcă pagina

Deschide tab-ul Network din DevTools:

// Tab Network arată TOATE request-urile:
// - HTML, CSS, JS, imagini, fonturi
// - API calls (fetch, XMLHttpRequest)
// - Timpii de încărcare
// - Response body (ce a returnat serverul)

// Fă un fetch direct din consolă și vezi-l în Network:
fetch("https://jsonplaceholder.typicode.com/users/1")
    .then(r => r.json())
    .then(data => console.log("User:", data.name, data.email))
