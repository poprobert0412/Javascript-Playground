Exercițiu 1 ⭐⭐⭐⭐

➕ Calculează suma unui array

Scrie o funcție suma care primește un array de numere și returnează suma lor.

Folosește return numere.reduce((total, n) => total + n, 0);
Exercițiu 2 ⭐⭐⭐⭐

🔍 Filtrează numerele pare

Scrie o funcție numPare care returnează doar numerele pare dintr-un array.

return numere.filter(n => n % 2 === 0); — % returnează restul: 4%2=0 (par), 5%2=1 (impar)
Exercițiu 3 ⭐⭐⭐⭐

🏗️ Creează un obiect

Creează un obiect masina cu proprietățile: marca, model, an, și o metodă descriere().

descriere() { return `${this.marca} ${this.model} (${this.an})`; } — this se referă la obiectul curent
Exercițiu 4 ⭐⭐⭐⭐

📦 Variabile — let vs const

Declară scor cu let (valoare 0), incrementeaz-o de 3 ori. Declară nume cu const. Afișează ambele.

let scor = 0; scor++; scor++; scor++; const nume = "Robert"; — scor va fi 3!
Exercițiu 5 ⭐⭐⭐⭐

🏷️ typeof — Ce tip are?

Declară 5 variabile de tipuri diferite și afișează typeof pentru fiecare.

console.log("numar:", typeof numar); — repetă pentru fiecare. typeof null = "object" (bug celebru!)
Exercițiu 6 ⭐⭐⭐⭐

✨ Template Literals

Declară nume, varsta, oras și construiește un mesaj cu backticks.

Completează cu datele tale: const nume = "Robert"; și ${nume} în template literal.
Exercițiu 7 ⭐⭐⭐⭐

🔢 Aritmetică și Modulo (%)

Calculează suma, diferența, produsul, câtul și restul împărțirii lui 17 și 5.

a / b = 3.4, a % b = 2 (rest: 17÷5 = 3 rest 2). Math.floor(a/b) = 3
Exercițiu 8 ⭐⭐⭐⭐

⚙️ Funcții — Return vs No Return

Scrie 2 funcții: una care returnează aria dreptunghiului, una care doar afișează un mesaj.

return lungime * latime; vs console.log(text); — prima RETURNEAZĂ, a doua AFIȘEAZĂ
Exercițiu 9 ⭐⭐⭐⭐

➡️ Arrow Functions

Rescrie funcțiile clasice ca arrow functions cu forma scurtă.

const dubluArrow = (n) => n * 2; — fără {} și fără return pe o linie!
Exercițiu 10 ⭐⭐⭐⭐

🔀 if / else — Verifică un număr

Scrie o funcție care returnează "pozitiv", "negativ", sau "zero".

if (n > 0) return "pozitiv"; else if (n < 0) return "negativ"; else return "zero";
Exercițiu 11 ⭐⭐⭐⭐

❓ Ternary Operator

Rescrie if/else folosind operatorul ternary ? :

const mesaj = varsta >= 18 ? "major" : "minor";
Exercițiu 12 ⭐⭐⭐⭐

🔄 For Loop

Afișează numerele 1-10 și calculează suma lor cu for loop.

suma += i; console.log(i); — += e scurtătura pt suma = suma + i
Exercițiu 13 ⭐⭐⭐⭐

🗺️ Array .map() — EUR → RON

Convertește prețuri din EUR în RON (1 EUR = 5 RON) cu .map()

preturiEUR.map(pret => pret * 5)
Exercițiu 14 ⭐⭐⭐⭐

🔍 Array .filter() — Doar majori

Filtrează din array-ul de vârste doar persoanele >= 18 ani.

varste.filter(v => v >= 18) — returnează true/false pt fiecare element
Exercițiu 15 ⭐⭐⭐⭐

📊 Array .reduce() — Coș cumpărături

Calculează prețul total din coșul de cumpărături cu .reduce()

cos.reduce((total, item) => total + item.pret, 0) — start de la 0, adaugă .pret
Exercițiu 16 ⭐⭐⭐⭐

🧱 Obiecte — Profil complet

Creează un obiect profil cu proprietăți, un array de hobby-uri, și o metodă prezentare().

prezentare() { return `${this.nume}, ${this.varsta}. Hobby: ${this.hobbyuri.join(", ")}`; }
Exercițiu 17 ⭐⭐⭐⭐

🔗 Chaining — .filter() + .map()

Filtrează studenții promovați (nota >= 5) și transformă-i în mesaje formatate.

.filter(s => s.nota >= 5).map(s => `${s.nume} — Nota: ${s.nota}`)
